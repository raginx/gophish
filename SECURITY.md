# Security Policy

## Reporting a Vulnerability

Thank you for taking the time to find and report a security vulnerability in this fork!

Please report it privately rather than opening a public issue, either via:

- [GitHub Security Advisories](https://github.com/raginx/gophish/security/advisories/new) for this repository (preferred), or
- email: reinhard [at] westerholt [dot] me

I'll work with you to get the vulnerability resolved as quickly as possible, and will credit you (if you'd like) once it's fixed.

If the vulnerability also affects [upstream Gophish](https://github.com/gophish/gophish), please consider reporting it there as well.
